# Definición de la granularidad de las visualizaciones

Ajuste fácilmente cómo se muestran los datos con el selector de granularidad en la configuración de visualización de tendencias. Esta función simplifica el análisis de datos al permitirle refinar las visualizaciones para adaptarlas a sus necesidades. Explore cómo optimizar los flujos de trabajo y hacer que las perspectivas de datos sean más accesibles.
