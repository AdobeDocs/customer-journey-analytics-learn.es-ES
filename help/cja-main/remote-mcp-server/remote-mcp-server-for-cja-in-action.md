# Servidor MCP remoto para [!DNL Customer Journey Analytics] en acción

Aprenda a conectar herramientas de IA como Claude, ChatGPT o Cursor al servidor MCP de Adobe Customer Journey Analytics (CJA) para explorar vistas de datos, ejecutar informes y generar perspectivas de KPI preparadas para ejecutivos utilizando lenguaje natural. Esta demostración muestra cómo los servidores MCP remotos habilitan flujos de trabajo de análisis escalables y controlados, así como sistemas de informes automatizados, en Adobe Experience Platform.

Para obtener más información, consulte la [documentación](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"} de Content Analytics.
