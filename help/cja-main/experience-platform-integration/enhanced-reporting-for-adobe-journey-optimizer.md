# Informes mejorados para Adobe Journey Optimizer

Hay nuevas funciones de medición de orquestación y creación de informes disponibles en Adobe Journey Optimizer. Estas mejoras en Journey Optimizer también incluyen una mayor interoperabilidad y valor ampliado con Customer Journey Analytics a través de fuentes de datos adicionales, personalización avanzada, manipulación de datos, análisis y mucho más. Para obtener documentación detallada, consulte [Introducción a la nueva interfaz de creación de informes](https://experienceleague.adobe.com/es/docs/journey-optimizer/using/channel-report/report-gs-cja){target="_blank"}.
