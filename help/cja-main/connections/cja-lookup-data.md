# Búsquedas de CJA para datos de perfil y búsqueda

Con las búsquedas de CJA en los datos de búsqueda y perfil, los usuarios pueden utilizar relaciones más profundas entre elementos de datos, como la persona a cuenta en casos de uso B2B.  En este vídeo, proporcionamos información sobre cómo añadir datos de búsqueda a conjuntos de datos de búsqueda y perfil, y sobre cómo planificar y asignar los campos clave.  Tenga en cuenta que si existe una relación de uno a muchos entre la clave y el valor de búsqueda, se utilizará el valor más reciente.

Para obtener documentación detallada, visite [Tipos de conjuntos de datos](https://experienceleague.adobe.com/es/docs/analytics-platform/using/cja-connections/create-connection#dataset-types){target="_blank"} y [Añadir datos de nivel de cuenta como un conjunto de datos de consulta](https://experienceleague.adobe.com/es/docs/analytics-platform/using/cja-usecases/b2b/example){target="_blank"}.
