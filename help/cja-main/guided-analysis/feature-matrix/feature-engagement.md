# Análisis de participación de funciones

Aprenda a usar la **vista de participación de funciones** en Customer Journey Analytics. Este análisis proporciona perspectivas sobre la frecuencia con la que se utilizan las funciones del producto frente al número de personas que las utilizan. Compare varias funciones al mismo tiempo para obtener ventajas óptimas. Esto ayuda a tomar decisiones de inversión al comprender sus características principales, su potencia, sus características únicas y cuestionables.
